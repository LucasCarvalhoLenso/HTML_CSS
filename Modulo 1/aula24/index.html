citações simples

para fazer citaçoes podemos usar a tag Q

citacões completas

usamos a tag

BLOCKQUOTE

para citar um determinado autor e para que a citaçao fique um pouco deslocada para dentro, tambem pode citar o local onde pegou a referencia(nao tem interaçao ou mudança nenhuma na parte visivel do site mas ajuda na parte das buscas) com o parametro CITE dentro do BLOCKQUOTE

abreviação

para marcar abreviações e mostrar seus significados passando o mouse por cima usamos a tag ABBR
ex: HTML e CSS

texto invertido

para inverter o texto usamos a tag BDO ela tem duas variaçoes sendo elas BDO:L(escreve o texto da esquerda pra direita) e BDO:R(escreve o texto da direita pra esquerda)
ex: Ola mundo! e Ola mundo!
