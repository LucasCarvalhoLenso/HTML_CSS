principais formataçoes

negrito e destaque

podemos colocar em negrito com a tag B(sem semantica)

ou podemos usar a tag STRONG(com semantica)

italico ou enfase

podemos fazer italico com a tag I(sem semanica)

ou podemos usara a tag EM(com semantica)

texto marcado

podemos marcar um texto com a tag MARK inclusive pode alterar a cor de um marca texto com um style e background-color dentro da tag mark

se quiser mudar a cor de todos de uma vez faz do mesmo jeito mas pelo css

texto grande ou pequeno

podemos usar a tag BIG para letras grandes(sem semantica) e a small para letra pequenas(com semantica)

texto deletado

podemos marcar um texto como excluido com a tag DEL

texto inserido

e tambem podemos marcar um texto como inserido com a tag INS

texto sobrescrito

para botar um texto mais acima podemos usar a tag SUP ex: x20=100

texto subscrito

esse para coisas mais abaixo usamos a tag SUB ex: H2O
