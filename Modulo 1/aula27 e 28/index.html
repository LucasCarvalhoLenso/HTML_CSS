links externos

para redirecionar nosso site podemos usar a tag a

entretanto voce pode usar parametros como target="_blank" para que inves de redirecionar o site esse link seja aberto em uma nova guia e rel="external para refericiar que o site é externo".
exemplo: mapa genshin

links internos

quando usamos link internos nao precisamos usar a url do site podemos apenas usar o nome do arquivo, podendo facilitar ainda mais apenas escolhendo o arquivo com o atalho ctrl+espaço
exemplo: armas

tambem podemos usar arquivos dentro de subpastas mas pra isso temos que indicar a pasta depois o nome do arquivo mas isso pode ser facilitado com o atalho ensinado acima
exemplo: personagens
